# sleodlx/maven-allure-chrome-stable
## Description
Docker image for Java based automated UI tests with chromedriver and allure report.

## Contained Libs
* openjdk 17
* Maven 3
* Allure CLi 2.4
* Chrome Stable (current 117)
* Chromme Driver (/usr/bin/chromedriver)
